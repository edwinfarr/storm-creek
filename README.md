# Storm Creek

Outdoor workout tracker PWA. Built for 50 acres in Caswell County, NC.

Dark. Foreboding. No backend. All local.

## Use it

Visit the deployed app and add to your home screen. All data stays on your device.

## Features
- 4 workout types: Ridge Run, Storm Fire, Dark Water, War Path
- 35 exercises (all knee/hip safe)
- Timer, rep counter, station-to-station transit
- FORGE warmup + RECEDE cooldown built into every session
- Coach Mode for wall-mounted displays
- Called exercises (micro-workouts throughout the day)
- Trend charts, PRs, calendar heatmap
- Offline-first PWA — works without internet
- Station presets (Storm Creek, Backyard, Park)
